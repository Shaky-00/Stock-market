### 金融数据爬虫项目

------------------------------------------

**项目描述**

​		从公开的新浪财经、东方财富等财经类网站爬取A股全市场每只个股的日线行情（包括高开低收价格以及成交量、成交额等统计数据）



**具体要求**

- 设计爬虫，在财经类网站爬取尽可能多的历史行情数据；部署实盘服务，每天定时爬取当天的数据。
- 在Jupyter notebook或Jupyter lab中对上述数据进行简单聚合分析
